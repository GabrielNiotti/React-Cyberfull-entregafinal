# 💻 CyberFull

> E-commerce desarrollado con **React**, **Vite** y **Firebase** como proyecto final.

CyberFull es una aplicación web orientada a la venta de productos tecnológicos. Permite a los usuarios explorar el catálogo, visualizar el detalle de los productos, administrar un carrito de compras y aplicar cupones de descuento. Además, cuenta con un panel de administración para gestionar el catálogo de productos.

---

## 🚀 Tecnologías utilizadas

- ⚛️ React
- ⚡ Vite
- 🔥 Firebase
- 🗄️ Firestore Database
- 🔐 Firebase Authentication
- 🧭 React Router DOM
- 🧠 Context API
- 🎨 CSS Modules
- 🖼️ ImgBB (almacenamiento de imágenes)
- 💬 SweetAlert2

## ✨ Funcionalidades

- Visualización del catálogo de productos.

## 🔐 Variables de entorno

El proyecto utiliza Firebase.

Crear un archivo `.env` con las credenciales correspondientes.

Ejemplo:

```env
VITE_API_KEY=xxxxxxxxxxxxxxxx
VITE_AUTH_DOMAIN=xxxxxxxxxxxxxxxx
VITE_PROJECT_ID=xxxxxxxxxxxxxxxx
VITE_STORAGE_BUCKET=xxxxxxxxxxxxxxxx
VITE_MESSAGING_SENDER_ID=xxxxxxxxxxxxxxxx
VITE_APP_ID=xxxxxxxxxxxxxxxx
```

> **Importante:** Las credenciales no están incluidas en este repositorio.
